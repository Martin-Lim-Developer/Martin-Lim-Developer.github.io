Project 1:

HUNGRY MATES

DESCRIPTION

A small team project to create a web application using Google Place's API to help users decide where to eat based on their current location. This is an app for those who has trouble deciding a place to go.

The app will use the user's current geo location and set radius to grab 3 random restaurants (preferences can be set). If the user still can't decide, then the app can further pick one out of the 3 results.


TECHNOLOGY USED

#ReactJS #Ant-Design #Google-API

I value design that acts as a means of expression. Good design should represents the clients core identity, be it representing the professionalism of a big company, or showcasing the character of an aspiring artist.


I enjoy thoughtful and aesthetically pleasing design for a myriad of use cases, from mobile apps to web and desktop UIs.


I use tools like Adobe Illustrator and Adobe XD to present my designs from wireframes, to mock-up prototypes before finally developing them. It is crucial to nail the concepts and identity of the app prior development.

I like to explore different tools and technologies from web frameworks like Angular and ReactJS to mobile frameworks like Flutter. I believe this helps me develop an array of tools that I can assess and use accrodingly to best fit the current project at hand.


I value clean code and maintainability. It is such a tricky thing to achieve especially if the project is massive, nevertheless I always try my best to write good code.
